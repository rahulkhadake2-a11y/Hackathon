# VendorIQ Demo Video Setup

## Video File Placement

Place your demo video file in this folder (`public/`) with the filename:

- **demo-video.mp4**

Optionally, you can also add a poster image:

- **video-poster.jpg** (thumbnail shown before video plays)

## Recommended Video Specifications

- **Format:** MP4 (H.264 codec)
- **Resolution:** 1920x1080 (1080p)
- **Aspect Ratio:** 16:9
- **Duration:** ~2 minutes
- **File Size:** Keep under 50MB for fast loading

## Demo Video Content (Based on VendorIQ)

The demo should showcase:

### Scene 1: Login Page (0:00–0:15)

- Show the VendorIQ login interface
- User enters credentials (admin@erp.com / admin)
- Click login with loading animation

### Scene 2: Dashboard Overview (0:15–0:50)

- KPI cards with animated values
- Line charts showing delivery trends
- Bar charts for vendor comparison
- Color-coded performance badges

### Scene 3: Vendor List Table (0:50–1:15)

- Searchable/filterable vendor table
- Hover effects on rows
- Sorting by different metrics

### Scene 4: Vendor Profile Page (1:15–1:45)

- Vendor details with KPI summary
- Historical performance charts
- Alerts and notifications

### Scene 5: Navigation & Logout (1:45–2:00)

- Sidebar navigation demonstration
- Logout flow

## Fallback Animation

If no video is provided, an animated fallback preview will automatically display, showing the app flow with mini-UI animations.
